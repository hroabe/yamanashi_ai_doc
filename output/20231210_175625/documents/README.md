# 「山梨の魅力をPythonで伝える: DjangoとPandasを使ってリアルタイム観光アプリを作ろう」

## 概要
この勉強会では、PythonのウェブフレームワークDjangoとデータ分析ライブラリPandasを使って、山梨県の観光地情報をリアルタイムで表示するウェブアプリの開発を行います。山梨県の観光地の魅力を伝えるとともに、Pythonの応用力を体験することが目標です。
