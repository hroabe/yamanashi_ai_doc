# "未来のAI農業 - 山梨を用いたFastAPIとPyTorchの組み合わせ"

## 概要
"この勉強会では、FastAPIとPyTorch、そして山梨という3つのキーワードを組み合わせ、未来のAI農業をテーマに取り組みます。具体的には、山梨の成長状況を写真から分析し、そのデータをFastAPIを用いてWebアプリケーションに組み込みます。目指すは、AIが山梨の最適な収穫時期を判定するシステムの構築です。"
